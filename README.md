# 星淵迴響 STAR: Abyssal Echoes

> 弦斷之處,它在聽。

一款純手工編寫、零依賴、離線可玩的科幻太空歌劇文字冒險 RPG。

**🎮 線上遊玩:[caf555.github.io/STAR](https://caf555.github.io/STAR/)**

---

## 這是什麼遊戲

在被遺忘的邊境星區「幽弦星區」,一名平凡的倖存者追查遠古文明的迴響,揭開橫跨三個紀元的封印真相——並在人類的陰謀、覺醒的機器與淵外之物之間,決定星區的命運。

- **5 章主線 + 3 段出身專屬序章**,劇情依你的抉擇分歧
- **3 出身 × 3 職業**自由組合創角
- **7 名可招募夥伴**,各有好感度系統與個人任務
- **回合制小隊戰鬥** + **艦船對戰**(距離帶機制)
- **星圖航行**、**科技樹與製造**、**陣營結盟系統**
- **四種結局**,依你全程的抉擇與羈絆決定
- 完整音效與氛圍音樂(Web Audio 即時合成,零音檔)

完整設計文件見 [`GAME_SPEC.md`](./GAME_SPEC.md)。

## 怎麼玩

打開 [caf555.github.io/STAR](https://caf555.github.io/STAR/),點「新遊戲」開始創角,或先玩「引擎示範場景」熟悉介面。

- **選項**:點擊選項推進劇情;部分選項標示屬性檢定(如「🧠智力 DC12」),成功機率由你的屬性與技能決定
- **數字鍵 1–9**:快速選擇對應選項
- **戰鬥**:回合制,依反應值輪流行動;可攻擊、使用技能、使用道具、防禦、換位
- **星圖**(右側面板「🗺 星圖」):在迴響號上開啟,選擇可跳躍的星系
- **小隊管理**(「⚔ 小隊」):出戰上限 2 人,其餘夥伴留守艦上提供被動加成;好感夠高時可在此「交談」觸發個人任務
- **科技/製造**(「🔬 科技/製造」):消耗資料核心研究科技樹,或用素材製造消耗品
- **存讀檔**:localStorage 自動存檔 + 5 個手動存檔槽,可匯出/匯入 JSON 跨裝置備份

設定面板可調整打字機效果、字級、背景動效、音效與音樂開關與音量。

## 技術特色

- **純前端、零依賴**:Vanilla JS(ES2020+),無建置工具、無框架
- **引擎與資料分離**:`js/engine/` 是不含任何劇情的通用引擎(狀態管理、敘事渲染、檢定、戰鬥、艦船戰、星圖、商店、科技、音效);`js/data/` 是純資料的劇情內容,以宣告式的條件/效果 DSL 描述
- **離線可玩**:雙擊 `index.html` 即可執行,資料以 `<script>` 標籤載入(避開 `file://` 下的 CORS 限制)
- **GitHub Pages 部署**:`main` 分支即線上版本來源

```
index.html            # 遊戲殼層
css/main.css           # 科幻 HUD 主題樣式
js/engine/             # 引擎(state / scene / check / combat / shipCombat /
                       #       starmap / shop / tech / audio / create / ui / core)
js/data/               # 劇情內容(世界觀 / 出身職業 / 各章節 / 夥伴 / 敵人 / 結局)
test.html              # 引擎斷言測試(數百項,含內容完整性 lint)
```

## 開發

沒有安裝步驟——用任何靜態伺服器(或直接雙擊)開啟 `index.html` 即可。

```bash
python3 -m http.server 8000   # 或任何靜態伺服器
# 瀏覽器開啟 http://localhost:8000/
```

跑引擎測試:瀏覽器開啟 `test.html`,頁面會列出所有斷言結果。
